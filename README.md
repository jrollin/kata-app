# kata-app

App to choose kata

## Stack

* [lit element](https://lit-element.polymer-project.org/)
* [babel](https://babeljs.io)
* [rollup](https://rollupjs.org/guide/en/)


## Install deps

```
npm install
```

## build 

```
npm run build
```


## run in local server

```
npm run start
```


## Tests 

used module @open-wc, see https://open-wc.org/init/

* [karma](https://karma-runner.github.io)
* eslint
* prettier
* husky



```
npm run test
```
